# Financial Health Dashboard

## Project Overview

This Power BI dashboard was developed as part of the CodeAlpha Power BI Internship.

The dashboard provides a comprehensive analysis of an organization's financial performance through interactive visualizations, financial statements, KPI monitoring, and forecasting.

## Features

* Total Sales Analysis
* Total Profit Analysis
* Units Sold Tracking
* Profit Margin Calculation
* Income Statement Summary
* Balance Sheet Summary
* Cash Flow Summary
* Profit Trend Forecasting
* Country-wise Sales Analysis
* Product-wise Profit Analysis
* Interactive Filters

## Tools Used

* Microsoft Power BI
* Power Query
* DAX
* Microsoft Excel

## Key Insights

* Identified profitable products and sales regions.
* Analyzed financial performance trends over time.
* Forecasted future profitability for budgeting and planning.
* Visualized financial statements for better decision-making.
